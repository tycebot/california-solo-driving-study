# California Solo Driving Study
Author: Terrell Enoru

Date of Submission: 12/9/2025

Structure: The project contains one folder named analysis. Analysis contains the quarto file containing the code,the paper produced by it, and the bib file with all the references used in the paper. In the base folder there is one notable file, this readme which provides a description of the repository structure and important information on the proejct.

## Data 

Data used in this project is obtained from the [California Open Data Portal](https://data.ca.gov/dataset/transportation-to-work) who provide the data under the [Creative Commons Attribution 4 License](https://creativecommons.org/licenses/by/4.0/legalcode).
